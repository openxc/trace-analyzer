This map and graphs are visualizations of data collected using the OpenXC Platform. OpenXC is an open souce software and hardware platform to pull data from your car and use it like any other data source in a smartphone, tablet or web application. For more information on OpenXC, visit the project page.

If you reached this page because of New York International Auto Show coverage, don't miss the Ford Personalized Fuel Efficiency App Challenge.

These visualizations are compatible with IE9+, Google Chrome, Safari, and Mozilla Firefox. The map overlays can be toggled via the layers button.
